JS

Java Script é a linguagem responsável por dar comportamento e interatividade às páginas web, permtindo responder a ações do usuário (como cliques e digitação), manipular o conteúdo do HTML em tempo real e se comunicar com servidores;

Em resumo, enquanto HTML estrutura e CSS estiliza, o JavaScript faz a página "funcionar".
